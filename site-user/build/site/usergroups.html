User groups

We're living in a virtual world, but it's great from time to time to be able to meet in real life, chat with other Groovy users, discover new aspects of the language or platform, attend presentations about special features or ecosystem projects, and more. Groovy user groups are there for you to exchange with others about your favorite language.
