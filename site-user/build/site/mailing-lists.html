Mailing-lists

The Groovy mailing lists are a great way to interact with the Groovy developers and Groovy users.

Available lists

users@groovy.apache.org

General purpose list for questions and discussion about Groovy		BrowseSubscribeUnsubscribeArchive

dev@groovy.apache.org

More focused list about the language implementation and its evolution		BrowseSubscribeUnsubscribeArchive

commits@groovy.apache.org

Tracks all commits		BrowseSubscribeUnsubscribeArchive

notifications@groovy.apache.org

Contains JIRA and Github notifications		BrowseSubscribeUnsubscribeArchive

Alternative mailing list archives

  • The mailing lists are archived on MarkMail. You can browse recent emails.

    MarkMail also provide RSS feeds: users dev commits notifications

  • The mailing lists are archived on Nabble: combined users dev commits notifications

    You can even post messages to a mailing list through Nabble but you first need to be subscribed to the appropriate list!

  • You can also find archives on mail-archive.com: users dev commits notifications
  • You can also find some historic dev mailing list archives on MARC: groovy-dev
